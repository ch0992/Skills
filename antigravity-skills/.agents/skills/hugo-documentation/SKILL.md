# Hugo Documentation Skill

This skill provides instructions and tools for creating, organizing, and managing content in a Hugo-based project.

## Core Capabilities
- Creating new content files with proper archetypes.
- Managing front-matter metadata.
- Using project-specific shortcodes.
- Validating content structure and links.

## Directory Structure
- `content/`: Where all the markdown files reside.
- `archetypes/`: Templates for new content.
- `assets/`: Resources like images and scripts.

## Usage Instructions
1. Use the `hugo new` command to create new content.
2. Ensure you specify the correct section (e.g., `posts`, `docs`).
3. Fill in the front-matter fields according to the project's standards.
