# claude-skills 작성 표준

Claude Code가 트리거 키워드를 감지해 자동으로 실행하는 스킬.

---

## frontmatter 스키마

```yaml
---
name: [kebab-case, 필수]
description: >
  [스킬이 하는 일 한 줄 설명].
  사용자가 "[트리거1]", "[트리거2]", "[트리거3]" 등을 언급할 때 반드시 이 스킬을 사용한다.
  [Claude가 이 스킬로 해야 하는 핵심 행동 한 줄].
---
```

**규칙:**
- `description`에 트리거 키워드를 명시적으로 나열해야 Claude가 자동 인식함
- "반드시 이 스킬을 사용한다" 문구 포함 필수
- 트리거는 사용자가 실제로 쓸 법한 자연어 표현으로

---

## 본문 구조

```markdown
# [스킬 이름]

[이 스킬이 다루는 컨텍스트/배경 1~3줄]

---

## 워크플로우 / 작성 절차

[단계별 실행 흐름. 번호 목록 또는 다이어그램]

---

## [핵심 섹션 1 — 스킬에 따라 다름]

[세부 규칙, 형식, 예시]

---

## 프로젝트 참고

- **GitHub**: owner=`...`, repo=`...`
- **경로**: `경로/예시`
- **기타 프로젝트별 설정값**
```

**규칙:**
- 본문은 Claude가 컨텍스트 없이도 실행 가능한 수준으로 작성
- "프로젝트 참고" 섹션에 프로젝트 종속 값을 모아서 명시
- 코드 예시는 핵심 부분만, 전체 파일 인용 금지
